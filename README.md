# Fraud-Detection-in-Insurance
Astha Nagargoje (CS23168) , Sakar Karnewar (CS23169)

📌 Project Overview

Insurance fraud is a significant issue that causes billions of dollars in losses every year. Detecting fraudulent claims manually is time-consuming and inefficient due to the large volume of insurance data.

This project implements multiple Machine Learning classification models to automatically detect fraudulent insurance claims using the Insurance Claims Dataset. The system analyzes policyholder and vehicle-related information to classify whether a claim is fraudulent or genuine.

🎯 Objectives

The main objectives of this project are:
Detect fraudulent insurance claims using machine learning techniques
Implement and compare multiple classification models
Analyze model performance using evaluation metrics
Visualize results using confusion matrices and comparison graphs
